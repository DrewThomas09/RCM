# RAG source — reading a Provider X-Ray as investable evidence (and what not to infer)

X-Ray turns CMS public data into **diligence evidence**, not an investment
decision. Use it to frame questions, not to answer the IC.

**The diligence signals** (green / amber / red / gray) are each traceable to a
component the report shows — never a hidden composite:
- **Quality position** — from the peer-relative evidence index (top / mid /
  bottom tier). Green ≥ 66, amber 34–65, red < 34, gray when unrated.
- **Peer sample** — robust (≥30) / usable (≥5) / insufficient (<5 → gray).
- **Enforcement / staffing** (SNF) — SFF or abuse icon → red; penalties /
  low staffing / recent ownership change → amber; surfaced **separately**,
  never averaged into the quality index.
- **Local competition** — locality HHI (composition proxy, not market share).
- **Ownership context** — largest ownership type + ownership-count HHI.
- **Missing-data risk** — provider + state-peer missingness.

**When a signal is investable.** When it is (a) directly observed (not a
proxy), (b) backed by an adequate peer sample (n ≥ 5), (c) consistent with the
provider's other metrics, and (d) confirmed against the source. A single
red flag on a thin sample is a *question*, not a conclusion.

**What NOT to infer.**
- **Not commercial revenue** — except HCRIS hospital cost-report fields, no
  vertical shows revenue, payer mix, or private-pay volume.
- **Not market share** — concentration is provider-count composition.
- **Not causation** — percentile gaps are deviation, not impact; X-Ray makes
  no prediction (the data is single-snapshot — see the prediction-readiness
  audit).
- **Not a recommendation** — X-Ray never says buy/avoid.

**Better diligence questions.** "Which signal is observed vs proxy?" · "Is the
red flag explained by case-mix?" · "What management evidence would confirm
it?" · "Is the peer sample large enough to trust this percentile?" · "What's
missing that would change the read?"
