# Colorado CIVHC — Cost of Care

**Source:** CIVHC / CO APCD public-use file (April 2023 release).
**Geography:** Colorado — statewide + by DOI region (Denver, Boulder, Colorado
Springs, Pueblo, Ft. Collins, Grand Junction, Greeley, East, West).
**Years:** 2017–2021 (+ "All").
**Fields:** year, payer type (Commercial / Medicaid / Medicare / Medicare
Advantage / CHP+ / All), category, claim type (Professional / Outpatient /
Inpatient / Long-Term Care), DOI region → **total spend, member months,
per-person-per-year (PPPY)**; plus an outpatient-category detail sheet.
**Powers:** `/cost-structure` (CONTEXTUAL Colorado cost panel), Colorado market
intelligence.

**What it indicates:** all-payer market spending levels per person by region,
payer, and service type in Colorado.

**What it does NOT prove:** it is **not** a specific facility's operating
expense or revenue, and not national. It is market context.

**Diligence use cases:** frame a Colorado target's cost environment; compare
regional PPPY; understand payer-mix cost dynamics.

**Caveats:** all-payer aggregate (not provider-level); Colorado-only; PPPY =
spend / member-months. Attach a hospital CCN for real HCRIS facility opex.

**Suggested questions:**
- "Is PPPY a facility cost or market context?" (market context)
- "Can I compare this to a non-Colorado provider?" (only with a caveat)
- "Which payer/region has the highest per-person spend?"
