# Colorado CIVHC — Alternative Payment Models (APM)

**Source:** CIVHC / CO APCD public-use file (FY2026).
**Geography:** Colorado (all-payer market level).
**Years:** 2022–2024.
**Fields:** payer type (Commercial / Medicare Advantage / Medicaid / CHP+ /
Total / Unknown), year, metric (Primary Care Spending / Total Medical
Spending), **% APM, % FFS**, APM/FFS spend, HCP-LAN category 2A–4N breakdown,
integrated-systems and value-based-payment dimensions.
**Powers:** `/cms-apm` (LIVE Colorado APM Adoption section), `/payer-stress`
(CONTEXTUAL payer-pressure).

**What it indicates:** the share of Colorado medical spend flowing through
alternative payment models vs fee-for-service, by payer and year — i.e. how far
value-based care has penetrated each payer.

**What it does NOT prove:** it is **market-level, not provider-specific** APM
participation, and **Colorado-specific — do not generalize nationally.**

**Diligence use cases:** gauge value-based-care exposure of a Colorado target's
payer mix; track APM trajectory by payer.

**Caveats:** Unknown payer has no value (shown "—", never 0); ~17% missing on
some %/LAN fields where a denominator is excluded. PEdesk pins one well-defined
slice (Total Medical Spending, integrated-included) so each payer×year is
unique.
